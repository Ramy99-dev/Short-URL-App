To run locally : <br/>
<h4>Backend :</h4>
<ol>

<li> Run <code>npm install</code> to install all dependencies .</li>
<li> Configure .env file </li>

```
SERVER_PORT = 8000
DATABASE = database URL 

//for sending emails :
EMAIL = 
PASSWORD = 
//for JWT :
TOKEN_KEY = (base64 string)
```
<li> Run  <code>nodemon .</code> </li>
</ol>
